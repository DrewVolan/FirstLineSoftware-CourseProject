Курсовой проект. Волейко Андрей Владимирович. vandrewv16@gmail.com

Проект на ASP.Net Framework, web-приложение.
Данное веб-приложение может принять файл в форматах .txt и .docx, зашифровать или расшифровать (шифр Виженера) текст из этих файлов, либо введённый с клавиатуры текст, по заданному ключевому слову, введённому с клавиатуры. Также есть возможность сохранить результат в виде .txt или .docx файла. Основной функционал шифрования покрыт автоматическими Unit тестами.
